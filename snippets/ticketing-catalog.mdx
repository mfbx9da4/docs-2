An object is missing? Or a platform you want to integrate?

1. [Ask the Panora community for this integration.](https://panora.canny.io/integration-requests)
2. Browse existing requests to check if someone else asked for the Ticketing or feature you need. If so, upvote the existing request to show your support. You can also tip a contributor for this specific task!
3. If your desired integration isn't listed, click the "New Integration" button to submit a new integration request. Provide as much detail as possible about the Ticketing and the feature needed.
   Thanks! 🙏
